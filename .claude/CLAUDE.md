# Trabalho de RAID — Contexto do Projeto

## Sobre o projeto
Apresentação HTML de slides sobre RAID (Redundant Array of Independent Disks) para disciplina de Redes e Sistemas.

## Arquivo principal
`apresentacao-raid.html` — arquivo único, sem dependências externas além das imagens PNG locais e Google Fonts.

## Imagens disponíveis
- `RAID 0.png` — striping (dois discos)
- `RAID 1.png` — mirroring (dois discos)
- `RAID 3.png` — paridade dedicada (quatro discos)
- `RAID 4.png` — paridade em bloco dedicada (quatro discos)
- `RAID 5.png` — paridade distribuída (quatro discos)
- RAID 2, 6, 10 usam SVG inline

## Design system
- Fonte: Inter + Outfit (Google Fonts)
- Background: #f7f6f2 (off-white quente)
- Ink: #0a0a0a
- Accent: #f59e0b (amber)
- Radius: 12px
- Estilo de referência: https://cronograma-atlas.vercel.app/tese.html
