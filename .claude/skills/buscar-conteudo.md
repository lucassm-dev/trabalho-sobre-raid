- RAID 10: 35% dos deployments enterprise Linux
- RAID 6: 22% do mercado (arrays > 8TB)
- RAID 5: caiu para 12% (era 30% em 2020)
- Mercado NAS global: $34.5 bilhões (2024)
- 60-40 ratio software RAID vs hardware RAID

## Como integrar conteúdo novo
1. Pesquisar com WebSearch usando termos específicos
2. Verificar a fonte (preferir sources técnicas/acadêmicas)
3. Resumir em 1-2 bullet points
4. Adicionar ao slide correto usando o skill `criar-slide`
5. Citar a fonte no contexto se necessário
